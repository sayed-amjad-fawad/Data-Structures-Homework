# Data Structures Homework

This repository contains my weekly homework assignments and code implementations for the Data Structures course (Fall 2025).

## Homework Contents:
* **Week 01:** Array operations (insertion and deletion logic).
* **Week 02:** Big O Notation and complexity analysis homework.
* **Week 03:** Singly Linked List implementation homework.
* **Week 04:** Doubly Linked Lists and recursive deletion tasks.
* **Week 05:** Shunting Yard algorithm (infix to postfix) homework.
* **Week 06:** Binary Trees and pre-order traversal logic homework.
* **Week 11:** Graph Clustering methods homework.
